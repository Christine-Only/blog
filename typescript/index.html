# TypeScript介绍

# 为什么要使用TypeScript?

TIP

简单来说就是因为JavaScript是动态类型，很多错误只有在运行时才会被发现，而TypeScript提供了一套静态检测机制，可以帮助我们在编译时发现错误。

# 搭建TypeScript学习环境

注意

TypeScript版本：V4.5.3

JavaScript版本：ES2020

# 安装最新版TypeScript

npm i -g typescript

# 安装ts-node

npm i -g ts-node

# 创建一个 tsconfig.json 文件

tsc --init

然后新建index.ts，输入相关练习代码后，执行ts-node index.ts即可。
